🚧 Em breve

O checklist do Mercado Livre está sendo preparado.
Assim que estiver pronto, vai aparecer aqui.

0 de 5 etapas concluídas0%
1. Criar a conta do cliente
Acessar o link de cadastro: app.upseller.com/register
Criar a conta usando o mesmo Gmail e senha já utilizados nos marketplaces do cliente.
Enviar o código de verificação para o e-mail do cliente e confirmar.
2. Integrar todos os marketplaces
Ir em Configurações (canto superior direito) → Integração de Loja.
Selecionar cada marketplace desejado → Conectar Loja.
Preencher o nome da loja do cliente e conferir se o marketplace correspondente está certo ao lado.
Repetir para todos os marketplaces do cliente.
3. Configurar notas fiscais
Voltar em Configurações → Configurações de Notas Fiscais.
Inserir o CNPJ do cliente, o certificado digital e a senha do certificado.
Salvar os dados fiscais.
4. Ativar impulsionamento de produtos (Shopee)
Ir em Produtos → localizar Shopee → aba Marketing.
Na lateral esquerda, clicar em Impulsionando Produto → Configurações → Adicionar Produtos → Salvar Mudanças.
Voltar e ativar o botão de Impulsionar.
5. Personalizar etiquetas de envio Quando necessário
Ir em Configurações → Configurações de Envio.
Selecionar o marketplace desejado → Configurações de Etiqueta.
Escolher o formato 10x15 ou A4 Sulfite, conforme a impressora do cliente.
0 de 7 etapas concluídas0%
1. Acessar o GPT Diretor Criativo
Clique no link para abrir o Diretor Criativo Mercado Livre no ChatGPT: Diretor Criativo (ChatGPT)
2. Enviar a foto e as informações do produto
Enviar a foto real do produto que precisa ser editado, junto com as informações e especificações completas (nome, material, tamanho, cor, uso etc.).
O GPT vai fazer a leitura do produto e montar o roteiro do funil de 8 fotos.
3. Aprovar o prompt da imagem
Para cada imagem do funil, o GPT vai montar o prompt e pedir para digitar "ok" para confirmar.
4. Gerar a imagem
Depois de digitar "ok", digite "gerar" para o GPT criar a imagem daquela etapa.
5. Repetir até a imagem 8
Repita o ciclo "ok" → "gerar" para cada imagem do funil. O GPT monta o próximo prompt automaticamente após cada imagem gerada, até completar as 8 imagens.
6. Gerar o título
Após a imagem 8, peça: "gerar título mercado livre" ou "gerar título shopee", conforme o marketplace do anúncio, para o GPT criar o título no padrão correto.
7. Gerar a descrição
Peça "gerar descrição" para o GPT finalizar o anúncio completo.
